The CAPH framework is now only provided as source code.


Building the compiler from source distribution requires Objective Caml and a C/C++ compiler (preferably GCC). Building the GUI additionnally requires Qt (vers >=5.2).


Some extra packages may be needed (they are not part of the Caph distribution)

  1. -to visualize the data-flow graphs generated by the compiler, you will need the graphviz package.

  2. -for compiling the SystemC generated code, a working SystemC installation is required

  3. -for simulating and synthetizing the VHDL generated code, a simulation and synthesis tool chain (we use GHDL for simulation and Quartus from Intel for synthesis). Float support for the VHDL backend requires the  FPHDL package
