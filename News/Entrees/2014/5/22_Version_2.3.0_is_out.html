Version 2.3.0 is out

 

  1. support for bidimensional arrays (arrays of arrays, aka 1Dx1D arrays)

  2. [map] and [foldl] higher-order wiring primitives for describing parallel replication and reduction graph patterns (see Sec 2.4.5 of the reference manual); a nice example of how bidimensionnal arrays and map higher-order wiring can be combined is given in examples/working/filter_bank.

  3. reformated reference manual (cleaner presentation and a logo !)

  4. renamed actors [d], [dp], [f] and [fp] in the standard preludeto [dl], [dkl], [fl] and [fkl] resp. to minimize the risk of name clash with user code
